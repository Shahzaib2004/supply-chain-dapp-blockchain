# Shahzaib Supply Chain DApp

Blockchain supply chain management on Ethereum Sepolia.

## Setup

```bash
# Backend
npm install
npx hardhat test

# Deploy
npx hardhat run scripts/deploy.js --network sepolia

# Frontend
cd frontend && npm install && npm run dev
```
